Circular Queue Sim

Based on a traffic simulator I once saw, this sim explores how work flow is impacted by random fluctuations in the rate of each process.

The grey circles are stores with initially the same number of items in each. The blue circles are processes that move one item between adjacent stores. If a store has no stock to serve a process, it flashes white.

Each process duration ranges randomly from 300 to 600ms.
