Galton Board

This simulation models a simple Galton Board setup, showing how the probabilities of a ball bouncing randomly left or right lead to something resembling a bell curve distribution. I dreamed this project up to try out different animation approaches.

The Race #1

This simulation models a parallel operation network with one shared resource. Decide how often the resource needs to switch tasks, in order to maximise the throughput of the system.

Based on a TOC workbook scenario to highlight the impact of individual performance measures.

The Race #2

This simulation models a serial operation network with one shared resource. As above, decide how often the resource needs to switch tasks, in order to maximise the throughput of the system.

Stopwatch control

This simulation is more about how to control animations using user-input. It prototypes a Stopwatch component that displays itself, and responds to user-events. It can handle the timing of simulated tasks via CSS animation events.
