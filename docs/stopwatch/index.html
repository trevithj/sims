Stopwatch Control

I want to figure out a generic way to create a process timer component. That means a component that is easy to put on the page, that shows time updates that are not janky, and that can emit a "tick" event or similar.

So I have modified an old SVG animation of a stopwatch, so that it can render itself, and can be restarted or paused by user-input. And since it uses CSS animations, the "animationiteration" event is already there.

Duration: 500ms

Each iteration of the animation rotates the clock hand once. A listener updates the tick count, and displays the event's elapsed time. Restarting the animation automatically resets the elapsed time. The count is independent of the event in this case.
